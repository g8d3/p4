# ag-02 — Podcast scriptwriter

## Goal

Write a creative podcast script for 2 personas discussing practical applications of the p4 multi-agent file-based system.

## Task

Write `script.md` with:

### Format

- 2 personas: one tech-savvy, one curious beginner
- Conversational, not lecturing
- Duration: ~3-4 minutes
- Language: Spanish (Colombian voice TTS)
- Warm, engaging tone — explain concepts through real-world examples

### Topics to cover

1. What is p4? (simple: a system where AI agents communicate through files and folders)
2. Real application 1: **Education** — create automated lessons, tutorials, Q&A for any subject
3. Real application 2: **Content creation** — massive video/image/text generation with multiple agents
4. Real application 3: **Public communications** — press releases, social media, customer responses
5. Why this matters: no complex infrastructure, agents work with simple files
6. Closing: how to get started (github.com/g8d3/p4)

### Structure example

```
[Intro music]
A: "¿Has oído hablar de p4?"
B: "¿Otro framework más?"
A: "No, es diferente... imagina agentes que se pasan mensajes con archivos"
...

[Each topic gets 30-45s of back-and-forth]
```

### Output

- `script.md` — full script with dialogue lines and timing cues
- Each line should indicate which persona speaks (A or B)

### Dependencies

ag-03 needs your script to generate TTS audio for each persona.
