# e001 — Test AGENTS.md

**Objetivo**: Probar y validar el AGENTS.md raíz creando un video que explique el sistema multiagente basado en archivos y directorios.

## Qué hace

Este experimento produce un video (9:16, vertical) que explica el concepto:
- Sistema multiagente sin base de datos ni buses de mensaje
- Comunicación entre agentes vía AGENTS.md anidados y estructura de directorios
- Comparación con enfoques tradicionales
